## 2일차
1. 파이썬 기본
    - 변수
    - 자료형
    - 연산자

```python
# 변수
val = 1

# 자료형
print(type(val))    # <class 'int'>

# 문자열 포맷팅
pi = 3.141592
print(f'pi는 {pi}입니다.')              # pi는 3.141592입니다.
print(f'pi는 {pi:0.2f}입니다.')         # pi는 3.14입니다.
print(f'pi는 {pi:10.3f}입니다.')        # pi는      3.142입니다.
```
